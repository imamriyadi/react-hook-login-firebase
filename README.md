
Simple Login Dengan React hooks

Clone repository <br>
`git clone https://github.com/imamriyadi/react-hook-login-firebase.git`<br>

Masuk Direktory <br>
`cd react-hook-login-firebase`

## Available Scripts

Install Library:

### `npm install`

Menajalankan Project:

### `npm start`

Setting Config Firebase 

### `/src/config/firebase.js`
